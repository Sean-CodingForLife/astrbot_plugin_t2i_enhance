# Changelog

## v2.2.0 - 2026-07-06

- 将插件增强配置从“全局单份配置”改为“按官方模板名绑定的配置组”。
- 新增 `template_profiles`，通过官方支持的 `template_list` 在 WebUI 中管理多组模板绑定配置。
- 当前激活模板切换后，插件会自动切换到同名绑定配置，不再让不同模板共用一套增强参数。

## v2.1.0 - 2026-07-06

- 删除插件内“官方模板 / 插件模板来源切换”配置。
- 改为插件启动后直接接管当前 AstrBot 已启用的 T2I 模板。
- 插件不再重复维护官方模板名与内联模板来源设置。
- 保留增强渲染所需的变量注入、背景图、日期时间、HTML/Markdown 处理和截图选项。
- 重写 README 和配置说明，使插件职责回到“增强当前激活模板”这一条主线。

## v2.0.0 - 2026-07-06

- 推倒重建插件架构，统一改为基于 `html_render(template, data, options)` 工作。
- 插件直接接管符合条件的 T2I 渲染，不再保留旧式文本预处理路线。
- 增加后端变量注入能力，支持正文 HTML、背景图、日期、时间、自定义变量等数据进入模板。
- 增加截图参数配置，用于控制 `html_render` 的截图行为。
- 重写配置、元数据和文档，使其全部围绕新架构组织。
